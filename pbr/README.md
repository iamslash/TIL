# Key Elements

- GI(Global Illumination, 전역조명) : IBL(Image Based Lighting, 이미지기반라이팅)
- Energy Conservation (에너지 보존법칙)
- Reflectivity (반사도) : Diffuse & Specular (디퓨즈 및 스펙큘러)
- Microsurface (미세면) : Roughness (러프니스)
- Frenel's Law (프레넬의법칙)
- Metalicity (금속성)

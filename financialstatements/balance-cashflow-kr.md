# 재무상태표 및 현금흐름표

## 재무제표 3대 보고서

| 보고서 | 영어 | 보는 것 |
|--------|------|--------|
| **손익계산서** | Income Statement | 일정 기간 동안 얼마 벌고 얼마 썼나 |
| **재무상태표** | Balance Sheet | 특정 시점에 뭘 가지고 있고 뭘 빚지고 있나 |
| **현금흐름표** | Cash Flow Statement | 실제 현금이 어떻게 들어오고 나갔나 |

---

## 재무상태표 (Balance Sheet)

특정 시점의 재무 상태를 나타낸다.

```
┌───────────────────────────────────┐
│          Assets (자산)             │
│                                    │
│  Current Assets (유동자산)          │
│    - Cash (현금)                   │
│    - Accounts Receivable (매출채권) │
│    - Inventory (재고)              │
│                                    │
│  Non-current Assets (비유동자산)    │
│    - PP&E (유형자산)               │
│    - Goodwill (영업권)             │
│    - Intangible Assets (무형자산)   │
│                                    │
├────────────────────────────────────┤
│       Liabilities (부채)           │
│                                    │
│  Current Liabilities (유동부채)     │
│    - Accounts Payable (매입채무)    │
│    - Short-term Debt (단기차입금)   │
│                                    │
│  Non-current Liabilities (비유동)   │
│    - Long-term Debt (장기차입금)    │
│                                    │
├────────────────────────────────────┤
│        Equity (자본)               │
│    - 자본금, 이익잉여금 등          │
│                                    │
└────────────────────────────────────┘

핵심 등식: Assets = Liabilities + Equity
```

### 주요 용어

| 용어 | 의미 |
|------|------|
| **Assets** | 자산. 회사가 보유한 모든 것 |
| **Liabilities** | 부채. 회사가 갚아야 할 것 |
| **Equity** | 자본. 자산 - 부채. 주주의 몫 |
| **Cash & Equivalents** | 현금 및 현금성 자산. 즉시 사용 가능한 자금 |
| **Debt** | 차입금. 은행 대출, 회사채 등 |
| **Goodwill** | 영업권. M&A 시 장부가치 초과 지급액 |

## 현금흐름표 (Cash Flow Statement)

실제 현금의 유입과 유출을 세 가지 활동으로 구분한다.

```
Operating Cash Flow (영업활동 현금흐름)
  + Net Income
  + D&A (감가상각비 다시 더함 — 현금 지출 아니므로)
  + Working Capital 변동 (매출채권, 재고 등)
  ─────────────────────
  = 영업에서 들어온 실제 현금

Investing Cash Flow (투자활동 현금흐름)
  - CapEx (설비 투자)
  - 인수합병 지출
  + 자산 매각 수입
  ─────────────────────
  = 투자에 쓴/받은 현금

Financing Cash Flow (재무활동 현금흐름)
  + 차입금 조달
  - 차입금 상환
  - 배당금 지급
  - 자사주 매입
  ─────────────────────
  = 자금 조달/상환 현금
```

### 주요 용어

| 용어 | 의미 |
|------|------|
| **Operating Cash Flow (OCF)** | 영업활동 현금흐름. 본업에서 실제 들어온 현금 |
| **CapEx** (Capital Expenditure) | 자본적 지출. 설비/인프라 구매 비용 |
| **Free Cash Flow (FCF)** | OCF - CapEx. 자유롭게 사용 가능한 현금 |

### Free Cash Flow가 중요한 이유

```
Free Cash Flow = Operating Cash Flow - CapEx
```

FCF는 회사가 영업으로 벌어들인 현금에서 설비 유지/확장 비용을 뺀 것이다.
이 돈으로 **배당, 자사주 매입, M&A, 부채 상환** 등을 할 수 있다.

```
예:
  Operating Cash Flow:  500억
  CapEx:               -200억
  ─────────────────────
  Free Cash Flow:       300억
    → 배당 100억
    → 자사주 매입 100억
    → 현금 보유 100억
```

## 손익 vs 현금흐름의 차이

순이익이 높아도 현금이 부족할 수 있고, 순이익이 낮아도 현금이 풍부할 수 있다.

| 상황 | 순이익 | 현금흐름 | 이유 |
|------|-------|---------|------|
| 외상 매출 많음 | 높음 | 낮음 | 매출은 인식했지만 현금은 안 들어옴 |
| 감가상각 큼 | 낮음 | 높음 | 비용으로 잡혔지만 현금 지출 없음 |
| 대규모 설비 투자 | 변동 없음 | 급감 | CapEx로 현금이 나감 |

그래서 **순이익과 현금흐름을 함께 봐야** 회사의 실제 재무 건강을 판단할 수 있다.

## 재무제표 3개의 연결

```
손익계산서                  현금흐름표                  재무상태표
─────────                ─────────                 ─────────
Net Income ──────────▶  OCF 시작점
                         + D&A
                         + WC 변동
                         = OCF
                         - CapEx ──────────▶  PP&E 변동
                         = FCF
                                              Cash 잔액 ◀── 현금흐름 합계
                         차입/상환 ──────────▶  Debt 변동
Net Income ──────────────────────────────────▶  이익잉여금 변동
```

세 보고서는 서로 연결되어 있다. Net Income은 현금흐름표의 출발점이 되고,
현금흐름의 결과는 재무상태표의 Cash와 Debt에 반영된다.

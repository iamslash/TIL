## 22.1 합성 대조법 (Synthetic Control)

대조군이 없을 때, **여러 비처치 그룹을 가중 합산**하여 "가상의 대조군"을 만든다.

```
예: 캘리포니아 담배세 효과
처치: 캘리포니아 (1988년 담배세 도입)
대조: 다른 주들의 가중 합산 → "합성 캘리포니아"
     (캘리포니아와 사전 추세가 비슷한 주들에 높은 가중치)

효과 = 실제 캘리포니아 - 합성 캘리포니아
```

DiD와의 차이: DiD는 평행 추세를 **가정**, 합성 대조는 사전 추세를 **맞추어** 만듦.

## 22.2 Double Machine Learning (DML)

기계학습으로 교란 변수를 통제하되, 인과 추정의 편향을 방지하는 방법.

```
1단계: ML로 X, Y 각각에서 교란 변수의 영향을 예측
2단계: 잔차끼리의 관계를 추정 → 인과 효과
```

고차원 데이터(변수가 매우 많을 때)에 유용.

---

## 22.3 회귀 킹크 설계 (Regression Kink Design)

RD의 변형: 처치가 기준점에서 **불연속적으로 변하는 것이 아니라**,
처치의 **기울기(강도)**가 변하는 것을 이용.

```
예: 소득세율이 특정 소득에서 10%→20%로 올라감
→ 기준점에서 세율 기울기가 꺾임 (kink)
→ 소득 반응의 기울기 변화 = 세율 변화의 인과 효과
```

---

## 22.4 Bunching 설계

기준점 주변에서 관측치가 **몰리는 현상**을 이용.

```
예: 세금 감면 기준이 소득 $50,000이면
→ $50,000 바로 아래에 사람들이 몰림 (소득 조작)
→ 얼마나 몰리는지가 세금 민감도를 나타냄
```

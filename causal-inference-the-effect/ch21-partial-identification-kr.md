# Chapter 21: 부분 식별 (Partial Identification)

> **핵심**: 점추정이 불가능할 때, **가능한 범위**를 제시한다

---

## 21.1 핵심 아이디어

지금까지의 방법: 가정을 세우고 → **하나의 숫자(점추정)**를 구함
부분 식별: 약한 가정만 세우고 → **범위(구간)**를 구함

```
점추정:   "기능 효과는 +24%이다"
부분 식별: "기능 효과는 +10% ~ +30% 사이이다"
```

**트레이드오프**: 가정이 약해질수록 → 범위가 넓어짐 → 하지만 더 믿을 수 있음

---

## 21.2 예제: 스포츠카와 과속

```
관찰: 스포츠카 소유자가 과속할 확률 +5%p 높음
교란: 위험 추구 성향 (관찰 불가)

가정 없이: 효과는 -∞ ~ +∞ (무의미)
약한 가정: "위험 추구는 스포츠카와 과속 모두에 양의 영향"
  → 통제하면 효과가 줄어들 것 → "효과 ≤ 5%p"
더 강한 가정: "위험 추구의 영향은 최대 3%p"
  → "효과는 2%p ~ 5%p"
```

---

## 21.3 언제 쓰는가?

- 핵심 교란 변수를 측정할 수 없을 때
- 다른 방법의 가정이 너무 강할 때
- 정책 결정에 "최소한 이 정도 효과는 있다"를 보여주고 싶을 때

---

## Python 예제

```python
import numpy as np

# 관찰된 효과 (교란 포함)
observed_effect = 5.0  # %p

# 가정: 교란의 편향이 0 ~ 3%p 사이
bias_min, bias_max = 0, 3

# 부분 식별 범위
effect_upper = observed_effect - bias_min  # 5.0
effect_lower = observed_effect - bias_max  # 2.0

print(f"관찰된 효과: {observed_effect}%p")
print(f"교란 편향 가정: {bias_min} ~ {bias_max}%p")
print(f"부분 식별 범위: [{effect_lower}, {effect_upper}]%p")
print(f"→ 스포츠카의 과속 효과는 2~5%p 사이")
```

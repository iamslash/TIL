# The Effect: An Introduction to Research Design and Causality

> **저자**: Nick Huntington-Klein
> **원서**: [theeffectbook.net](https://theeffectbook.net/)
> **이 문서**: 각 챕터를 한글로 요약. Junior Software Engineer도 이해할 수 있도록 작성.

---

## 이 책은 왜 읽어야 하는가?

A/B 테스트 결과를 해석하거나, 데이터 분석 리포트를 평가하거나, "이 기능이 정말
효과가 있었는가?"라는 질문에 답해야 할 때 필요한 **인과추론(Causal Inference)**의
기초를 다루는 책이다. 수식보다 직관과 예제 위주로 설명하며, 비전공자도 읽을 수
있다.

# Tips

| Short Cut | Descrption |
|--|--|
| Ctrl + w + w | Move Screens |
| Ctrl + w + = | Adjust Screens with same size |
| ] + c | 앞쪽으로 이동하며 차이점 찾기 |
| [ + c | 뒤쪽으로 이동하며 차이점 찾기 |
| d + p | 커서가 있는 쪽의 내용을 반대쪽에 복사 |
| d + o | 커서가 없는 쪽의 내용을 반대쪽에 복사 |
| z + o | 숨긴 부분 열기 |
| z + c | 차이점 없는 부분 열기 |
| :diffupdate | 차이점 비교 실행 |

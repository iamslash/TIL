# Abstract

* Method defer the exact steps of an algorithm to a subclass
* 추상 클래스에서 절차를 정의하고 구현 클래스에서 단위 절차들을 구현한다. template method 가 실행되면 실제 실행은 자식 클래스들로 지연되어 있다.

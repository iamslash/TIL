## Shorcuts

| ShortCut | Description |
|:--------|:---------|
| `^ L Click` | Quick Help |
| `⇧ ⌘ O` | Open Quickly |
| `^ ⌥ ⌘ Enter` | Assistant View on storyboard |
| `⇧ ⌘ L` | Library Window |
| `⌘ B` | Build |
| `⌘ R` | Run |

# Basic

## Shorcuts

| ShortCut | Description |
|:--------|:---------|
| `^ L Click` | Quick Help |
| `⇧ ⌘ O` | Open Quickly |
| `^ ⌥ ⌘ Enter` | Assistant View on storyboard |
| `⇧ ⌘ L` | Library Window |
| `⌘ B` | Build |
| `⌘ R` | Run |
| `^ ⌘ <-` | Back |
| `^ ⌘ ->` | Forward |
| `^ ⌘ J` | Jump to definition |

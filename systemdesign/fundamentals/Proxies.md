# What is the Proxy server ?

client 와 back-end server 사이의 중간 server 를 말한다.
filter requests, log requests, transform requests (by adding/removing headers, encrypting/decrypting, or compressing a resource)

# Proxy Server Types

* Open Proxy
* Reverse Proxy

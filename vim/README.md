# Tips

## Split windows

| Short cut         | Description                   |
| ----------------- | ----------------------------- |
| `:sp <file-name>` | split window with the file    |
| `^ww`             | move window                   |
| `^wt`             | move top window               |
| `^wb`             | move bottom window            |
| `^wj`             | move left window              |
| `^wk`             | move right window             |
| `^wc`             | close window                  |
| `^w_`             | max window                    |
| `20^w_`           | set window size with 20 lines |

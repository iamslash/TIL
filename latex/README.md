# Tips

* 한글 사용하기

```
\documentclass{article}
\usepackage[T1]{fontenc}
\usepackage{CJKutf8}
\usepackage[english]{babel}
\begin{document}
\begin{CJK}{UTF8}{mj}
한글이 됩니다.
\end{CJK}
\end{document}
```

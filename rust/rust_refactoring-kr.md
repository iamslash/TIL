# Rust 리팩토링 (Refactoring)

## 일반적인 리팩토링 패턴

### 메서드 추출
큰 함수를 작고 집중된 함수로 분해합니다.

### Clone을 참조로 교체
가능한 경우 복제 대신 차용을 사용합니다.

### 타입 매개변수 도입
재사용성을 위해 구체적인 타입을 제네릭으로 교체합니다.

### Trait 추출
공통 동작을 추상화하기 위해 trait를 생성합니다.

### 에러 처리 교체
`unwrap()`에서 `?`를 사용한 적절한 에러 전파로 마이그레이션합니다.

## 리팩토링 도구

* **rustfmt**: 자동 코드 포맷팅
* **clippy**: 린팅 및 제안
* **rust-analyzer**: 리팩토링 액션이 있는 IDE 지원

# magit

- emacs에서 git을 쉽게 사용할 수 있도록 도와주는 plugin이다.

# magit usage

```
s to stage files
c c to commit (type the message then C-c C-c to actually commit)
b b to switch to another branch
P u to do a git push
F u to do a git pull
try to press TAB
```

# intro

- 데이터 과학을 이해하려면 다음과 같은 주제들을 공부해야한다.
  - 데이터시각화, 선형대수, 통계, 확률, 가설과 추론, 경사 하강법, 기계학습
  - k-근접이웃, 나이브 베이즈, 단순 선형회귀, 다중 선형회귀, 로지스틱 회귀
  - 의사결정나무, 신경망, 군집화, 자연어처리, 추천시스템


# excel shortcut

| Category | shortcut | DESC |
| :------- | :------- | :--- |
| 셀입력    | CTRL + Enter | 여러 셀 데이터 입력 |
|          | CTRL + D | 위의 셀로 자동채우기 |
|          | CTRL + ALT + v | 선택하여 붙여넣기 |
|          | CTRL + ; | 오늘 날짜 입력 |
|          | CTRL + F3 | 함수입력 상자 |
| 이동, 선택 | CTRL + SHIFT + * | 데이터 블록만 선택 |
| | SHIFT + SPACEBAR | 전체 행 선택 |
| | CTRL + SPACEBAR | 전체 열 선택 |
| | CTRL + 방향키 | 맨 앞, 뒤 이동 |
| | CTRL + PgUp/PgDn | 시트 이동 |
| 기타 | CTRL + z | 되돌리기 |
| | CTRL + s | 저장하기 |
